Joseph Moore

An author's eCommerce site

Created by Krish Dholakiya / Colin McIllece

What It Is

This website is:

  • An eCommerce platform for Joseph Moore, an author, to sell copies of his book: Vision. Action. Motivation
  • A ticketing platform for Joseph Moore's book tour around America
  • A blog platform for Joseph Moore to post his thoughts in a lightweight but intuitive manner

Project Division

In order to complete tasks in an timely manner, creating different objectives for each of us was a necessity. The teamwork aspect was pre-planned in order to establish a better partnership and wireframe for the website.

    Krish's Work

  • Front-End Development - Jade, LESS, AngularJS
  • Infrastructure Management - Git Version Control, Heroku Platform-as-a-Service (PaaS), HarpJS Asset Pipelining
  • Backend Engineering - Firebase Backend-as-a-Service (BaaS)

    Colin's Work

  • Assist In Front-End Development
  • Graphic Design - Photoshop, Illustrator
  • Content Management - Jade, Markdown, Copywriting Methods
  • Information Architecture - Site Planning and Mockups
  • QA (Quality Assurance) - Coordination of Test-Driven-Development (TDD) Methodology

Front-End Development

  • Jade - A Whitespace-Sensitive Dynamic Templating Language for HTML
  • LESS - A CSS Preprocessor with variables, mixins, nesting
  • AngularJS - A Model-View-Controller (MVC) Framework for HTML and JavaScript

Infrastructure Management

  • Git Version Control System and GitHub Integration - A way to ensure all development code is consistent and streamlines collaboration
  • Heroku PaaS - VPS would be a hassle to manage w/o extensive time spent on sysadmin, shared hosting lack language support and robustness; a PaaS provides the best of both worlds
  • HarpJS - A way to ensure that all our Jade and LESS would be preprocessed without hassle, and allows for dynamic content powered by JSON files

Backend Engineering

  • Firebase BaaS - A cloud-based document-oriented backend with a AngularJS API, used to keep track of book and ticket orders; API ensured that data in the cloud perfectly synced with AngularJS $scope(like variables) in realtime

Graphic Design

  • The use of sophisticated graphic editing software such as Adobe Photoshop and Fireworks to imrpove and enhance every graphic on the website. Tools such as cropping, filters, and even fabricating images from scratch(book cover).

Content Management/Information Architecture

  • Content Management - Development of advanced grammar, deployed in such a way that the text will invite users to remain on a website longer and augment the buyer conversion rate(number of users who visit a site and buy a product or complete a task).
  • Information Architecture - Involves the categorization of information into a coherent structure, preferably one that the intended audience can understand quickly, if not inherently, and then easily retrieve the information for which they are searching. Aesthetics and usability are key to a successful website.

Quality Assurance (QA Development) and Test-Driven-Development (TDD)

  • Quality Assurance - The process of executing the functions on the website with the intent of finding software defects or bugs in order to improve the user experience and allow easier implementation of code.
  • Test-Diven-Development - A process involving the setup of code tests, execution of the tests, validation of the code, and cleanup of the code to ensure that only the bare minimum of code needed is included in the files.
