Joseph Moore

An author's eCommerce site

Created by Krish Dholakiya / Colin McIllece

What It Is

This site is:

  • An eCommerce platform for Joseph Moore, an author, to sell copies of his book Vision. Action. Motivation
  • A ticketing platform for Joseph Moore's book tour around America
  • A blog platform for Joseph Moore to post his thoughts in a lightweight but intuitive manner

Project Division

In order to complete tasks in an efficient manner, creating different objectives for each of us was a necessity.

    Krish's Work

  • Front-End Development - Jade, LESS, AngularJS
  • Infrastructure Management - Git Version Control, Heroku Platform-as-a-Service (PaaS), HarpJS Asset Pipelining
  • Backend Engineering - Firebase Backend-as-a-Service (BaaS)

    Colin's Work

  • Graphic Design - Photoshop, Illustrator
  • Content Management - Jade, Markdown, Copywriting Methods
  • Information Architecture - Site Planning and Mockups
  • QA (Quality Assurance) - Coordination of Test-Driven-Development (TDD) Methodology

Front-End Development

  • Jade - A Whitespace-Sensitive Dynamic Templating Language for HTML
  • LESS - A CSS Preprocessor with variables, mixins, nesting
  • AngularJS - A Model-View-Controller (MVC) Framework for HTML and JavaScript

Infrastructure Management

  • Git Version Control System and GitHub Integration - A way to ensure all development code is consistent and streamlines collaboration
  • Heroku PaaS - VPS would be a hassle to manage w/o extensive time spent on sysadmin, shared hosting lack language support and robustness; a PaaS provides the best of both worlds
  • HarpJS - A way to ensure that all our Jade and LESS would be preprocessed without hassle, and allows for dynamic content powered by JSON files

Infrastructure Management - Demo

Backend Engineering

  • Firebase BaaS - A cloud-based document-oriented backend with a AngularJS API, used to keep track of book and ticket orders; API ensured that data in the cloud perfectly synced with AngularJS $scope(like variables) in realtime
